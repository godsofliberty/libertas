Libertas

Libertas is a suite of cli tools intended for the twitter liberty activist, but files can be customized for your needs.

Download this project as a .zip file Download this project as a tar.gz file

Welcome to The Libertas Project!

libertas.py currently comes with the commands:

  • {tweet} "STRING"
  • {ffbot} with no options enters interactive mode
  • {ffbot} [-l, --list=LIST] [-s, --seconds=SECONDS] [-f, --file="FILE"] where list is the twitter list, seconds=wait between tweets, and "FILE" is the list of intros.
  • {ed} [-s, --seconds=SECONDS] "FILE" where SECONDS=wait between tweets, and "FILE"= file to tweet.
  • {list_check} "FILE" checks "FILE" line by line and counts characters, reporting a line with more than 140 characters. Good for checking intro and ed lists.

All optional options have set defaults.

Examples:

libertas.py ffbot -l 0 will set the twitter list to the first list you have on twitter and fill in the other options with defaults( --seconds=300 --file="data/intros.txt") This will split the twitter list into multiple lists of six users per list, and add the @ symbol to their names. Next, file intros.txt will be loaded, randomized and one will be selected and tweeted. Next the 300 forces libertas to wait 300 seconds after every tweet.

libertas.py was written in python 2.7 and requires the tweepy library tweepy-1.11.tar.gz or later.
