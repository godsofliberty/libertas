Libertas

Libertas is a suite of cli tools intended for the twitter liberty activist, but files can be customized for your needs.

Welcome to The Libertas Project!

Libertas is a program suite of tools for twitter.

  1. tweet is a command that simply tweets the string you supply on the command line.
  2. ffbot is an interactive #FF FollowFriday bot.
  3. ed is short for education. It is a bot similar to ffbot that will tweet a list of lines from a text file, such as quotes or facts.
  4. list_check will scan a file one line at a time for lines over 140 characters long, and report that line, which can then be edited.

Usage:

  • python libertas.py tweet "STRING" simply sends the string as tweet to twitter.

  • python libertas.py ffbot enters interactive mode, which will ask you to select the number corresponding to the list you want to tweet, then the wait in seconds between 30-3600 [default=300]. Next it will ask for the intros file you want to use. Once you press enter, ffbot will pick a random intro, break up the twitter list to smaller lists of six users and format them, and then tweet each one after the number of seconds specified.

  • python libertas.py ed works like ffbot but asks for the ed file to be tweeted instead. Both are now menu-driven. The options became complicated for the user.

  • python libertas.py list_check "FILE" checks "FILE" line by line and counts characters, reporting a line with more than 140 characters. Good for checking intro and ed lists.

libertas.py was written in python 2.7 and requires the tweepy library tweepy-1.11 or later.
