Libertas

Libertas is a suite of cli tools intended for the twitter liberty activist, but files can be customized for your needs.

Welcome to The Libertas Project!

Libertas is a suite of command line tools for twitter.

The current version of Libertas is 2.5.

Check Dropbox for the latest version of the Libertas Data files.

Libertas lets you:

  • Tweet from the command line. I do this alot!
  • Follow Friday to promote your followers easily.
  • Automate tweeting. Whatever you can put in a text file Libertas can tweet at regular intervals with ed. Facts, quotes, recipes, jokes--you name it--all can be automated 24/7.
  • Format your ed files to make sure your tweets are within Twitter's 140 character range.
  1. tweet is a command that simply tweets the string you supply on the command line.
  2. ffbot is an interactive #FF FollowFriday bot.
  3. ed is short for education. It is a bot similar to ffbot that will tweet a list of lines from a text file, such as quotes or facts.
  4. list_check will scan a file one line at a time for lines over 140 characters long, and report that line, which can then be edited.

  • python libertas.py tweet "STRING" simply sends the string as tweet to twitter.
  • python libertas.py ffbot requires you to have at least one Twitter list. The lists you have are displayed as a menu. Choose one. Tell Libertas at what interval to tweet (30 - 3600 seconds). Choose an intro. Hit Enter.
    • =======

  • python libertas.py tweet "STRING" simply sends the string as tweet to twitter.

  • python libertas.py ffbot enters interactive mode, which will ask you to select the number corresponding to the list you want to tweet, then the wait in seconds between 30-3600 [default=300]. Next it will ask for the intros file you want to use. Once you press enter, ffbot will pick a random intro, break up the twitter list to smaller lists of six users and format them, and then tweet each one after the number of seconds specified.

  • python libertas.py ed works like ffbot but asks for the ed file to be tweeted instead. Both are now menu-driven. The options became complicated for the user.

  • python libertas.py list_check "FILE" checks "FILE" line by line and counts characters, reporting a line with more than 140 characters. Good for checking intro and ed lists.

libertas.py was written in python 2.7 and requires the tweepy library tweepy-1.11 or later.

    Contribute!

    Anyone can contribute to Libertas Data. Just email me your data and I will update the Data distribution. You can contribute ed files on any subjects you like. You can add intros that are tweeted when the user runs certain tools, or exit tweets.

    If you know python you can contribute code.

    Here is the Dropbox link to the Data distribution. The Data version will change more frequently than the program.

    Data

    This zip contains the data directory with a README, changelog.txt and the directories, ed, intros, exits, and keys.

    If you contribute to the Data distrubution, or you haven't edited any of the files for your personal use, then a manual update is easy. Just back up the keys folder, overwrite the data folder and copy the keys back. If you have created an entry in an existing file, you might have to cut and paste. I will start an updater soon.
